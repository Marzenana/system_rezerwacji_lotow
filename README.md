# System rezerwacji lotów

## Opis

System rezerwacji biletów lotniczych `Cat Airline`.

## Instalacja zależności

`npm install`

## Uruchomienie wersji deweloperskiej projektu

`npm run start:dev`

## Budowanie projektu

`npm run build`
